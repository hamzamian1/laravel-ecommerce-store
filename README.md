Oxy-Bliss – E-commerce Website
- Project Description

Oxy-Bliss is a modern and fully functional e-commerce website inspired by premium brands. It provides a smooth shopping experience with advanced features like product variations (stitched/unstiched), cart system, secure checkout, and responsive design.

 Features
- Add to Cart & Wishlist system (fully functional)
- Product variations (Stitched / Unstitched with dynamic pricing)
- Order placement with unique Order ID
- User Authentication (Login / Signup)
- Admin Panel (Manage products, prices, orders, banners)
- Email notifications (order confirmation & updates)
- Search system (including product code/SKU)
- Hero banner with custom upload & adjustment
- Image hover effect (2 images per product)
- Fully responsive (Mobile + Desktop optimized)
- Newsletter subscription system
- Cookie consent system
- Order tracking with history
